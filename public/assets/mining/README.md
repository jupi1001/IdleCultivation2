# Mining assets

Mining uses **two kinds of images** in two places:

- **Area images** (this folder) = the **location card** (e.g. what the copper vein / cave looks like). Shown on the mining area card.
- **Item images** (`items/` subfolder) = **ore icons** (mining loot only). Small icons (e.g. 64×64). Bar icons live in the forging assets folder.

So e.g. `copper-mine.webp` in the root is the “Copper Ore” **location**; `items/copper-ore.webp` is the **ore** you receive.
